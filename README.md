# Curadentes
Solução para auxiliar dentistas, pacientes e prestadores de serviço se conectarem.

Consiste em uma aplicação voltada para o público geral (pacientes) encontarem especialistas e disponibilidade de maneira rápida, segura e eficiente, os dentistas terão acesso à histórico dos pacientes e organização de agenda, vendedores de insumos e prestadores de serviço, em um ecosistema seguro, limpo e organizdo.
